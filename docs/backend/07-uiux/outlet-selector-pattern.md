# Outlet Selector Pattern

## Owner/Admin

Show:

```txt
Outlet: All Outlets ▼
```

Dropdown:

```txt
All Outlets
SelaluTeh Samarinda
SelaluTeh Tenggarong
SelaluTeh Bontang
```

## Outlet Manager

If user has one outlet:

```txt
Outlet: SelaluTeh Samarinda
```

Can be locked/read-only.

## Rule

Only show outlets user can access.

Backend must still validate outlet access.
