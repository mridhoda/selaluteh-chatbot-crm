# PROJECT CONTEXT EXTRACTION PROMPT — Telegram Marketplace MVP

Kamu adalah AI engineering assistant yang sedang berada di dalam codebase project saya.

Saya ingin kamu melakukan **audit menyeluruh terhadap seluruh project ini** dan membuat dokumentasi teknis yang sangat lengkap, detail, dan terstruktur agar AI lain yang belum pernah melihat codebase ini bisa memahami project saya dengan cepat dan akurat.

Project ini akan saya lanjutkan menjadi:

> **Telegram-first Marketplace MVP dengan chatbot, product catalog, order flow, dan payment gateway sandbox.**

Saat ini project saya sudah memiliki dukungan chatbot ke Telegram. Saya ingin melanjutkan project ini, bukan membangun ulang dari nol.

Tugas utama kamu adalah membuat satu file Markdown bernama:

```txt
PROJECT_CONTEXT_REPORT.md
```

File ini harus berisi seluruh informasi penting tentang project, mulai dari struktur folder, teknologi yang digunakan, fitur yang sudah ada, fitur yang belum ada, flow chatbot Telegram, database, API, environment variable, deployment, masalah teknis, dan rekomendasi sprint berikutnya.

Jangan ubah kode aplikasi kecuali perlu membuat atau memperbarui file dokumentasi ini.
